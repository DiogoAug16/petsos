Frontend PetSOS

Aplicativo mobile construído com Expo Router e React Native. A interface prioriza navegação por tabs, mapa como primeira tela útil, ações com feedback tátil, telas públicas para convidados e permissões progressivas para usuários autenticados.

Autenticação

  • Login, cadastro, recuperação de senha e lembrar último identificador.
  • Envio de verificação de email pelo Firebase Auth.
  • Fluxos de convidado com acesso a denúncias públicas.

Mapa

  • Marcadores personalizados e popup antes do detalhe.
  • Rotas por ruas desenhadas no mapa do app.
  • Cache de tiles e prefetch de áreas próximas.

Comunidade

  • Comentários, respostas e reportes com motivo.
  • Acompanhamento de denúncias e notificações.
  • Voluntariado e evidências para validação.

Estrutura

Pastas principais

Otimizações no app

Percepção de velocidade

Cache local

  • Perfil do usuário e perfil público com stale-while-revalidate.
  • Contador e lista de notificações cacheados por usuário.
  • Detalhe de denúncia recente reaproveitado ao navegar.

Mapa e imagens

  • Tiles carregados em batch para reduzir requests individuais.
  • Marcadores renderizados com cache e invalidação realtime.
  • expo-image para cache de imagens e thumbnails em listas.

Comandos

Desenvolvimento
  • npm install instala dependências.
  • npm start inicia o Expo.
  • npm run ios abre no simulador iOS quando disponível.
  • npm run android abre no Android quando disponível.
  • npm run lint valida o código com Expo lint.
